# listOrderShippingMethods - Order Module Reference

This documentation provides a reference to the `listOrderShippingMethods` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method retrieves a paginated list of shipping methods based on optional filters and configuration.

## Example

To retrieve a list of shipping methods using their IDs:

```ts
const shippingMethods = await orderModuleService.listOrderShippingMethods({
  id: ["123", "321"]
}, {})
```

To specify relations that should be retrieved within the shipping method:

```ts
const shippingMethods = await orderModuleService.listOrderShippingMethods({
  id: ["123", "321"]
}, {
  relations: ["adjustments"]
})
```

By default, only the first `15` records are retrieved. You can control pagination by specifying the `skip` and `take` properties of the `config` parameter:

```ts
const shippingMethods = await orderModuleService.listOrderShippingMethods({
  id: ["123", "321"]
}, {
  relations: ["adjustments"],
  take: 20,
  skip: 2
})
```
