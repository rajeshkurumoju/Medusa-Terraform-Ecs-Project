# listPrices - Pricing Module Reference

This documentation provides a reference to the `listPrices` method. This belongs to the Pricing Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method is used to retrieve a paginated list of prices based on optional filters and configuration.

## Example

To retrieve a list of prices using their IDs:

```ts
const prices = await pricingModuleService.listPrices({
  id: ["price_123", "price_321"],
})
```

To specify relations that should be retrieved within the prices:

```ts
const prices = await pricingModuleService.listPrices(
  {
    id: ["price_123", "price_321"],
  },
  {
    relations: ["price_rules"],
  }
)
```

By default, only the first `15` records are retrieved. You can control pagination by specifying the `skip` and `take` properties of the `config` parameter:

```ts
const prices = await pricingModuleService.listPrices(
  {
    id: ["price_123", "price_321"],
  },
  {
    relations: ["price_rules"],
    take: 20,
    skip: 2,
  }
)
```
